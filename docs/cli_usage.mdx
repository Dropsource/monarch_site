To get the list of supported commands and arguments run:
```shell
monarch -h
```

### Notable flags
There are a couple of flags you can pass to `monarch run`:

1. `--delete-conflicting-output`: use it if you see any conflicting errors during code generation
2. `--no-sound-null-safety`: use it if your project has disabled sound null safety.
