cd development

    pathWindows: 'set PATH=%PATH%;C:\\path\\to\\monarch\\bin'

### Add dependencies
1. Add `monarch` and `build_runner` to your project dev_dependencies:
```yaml
dev_dependencies:
  monarch: ^0.0.32
  build_runner: ^1.10.3
```

2. Run `flutter pub get` on your project.

3. Create a build.yaml file at the top level of your project and add the 
following:
```yaml
targets:
  $default:
    sources:
      - $package$
      - lib/**
      - stories/**
```
